# Tuition Center App

A simple Flask-based tuition center application for managing students, courses, and enrollments.

## Getting Started

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
python app.py
```

3. Open your browser:

```
http://127.0.0.1:5000
```

## Features

- Add and list students
- Add and list courses
- Create enrollments linking students to courses
- Clean, responsive UI with a simple dashboard

## Files

- `app.py` — main Flask application
- `templates/` — HTML templates for the app UI
- `static/style.css` — app styles
- `requirements.txt` — Python dependencies
